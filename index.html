This approach accounts for options such as margins, padding, borders, and all the properties that manipulate them. The CSS Box Model is a term used for the container that wraps the following element properties within it.
  • Margin
  • Border
  • Padding
  • Content
  • Height and Width

We know that HTML tags are (mostly) used to format content - these tags tell the browser how to display the content on the page. They give no indication as to what type of content they contain or what role that content plays in the page. Semantic HTML5 addresses this shortcoming by defining specific tags to indicate clearly what role is played by the content those tags contain. That explicit information helps robots/crawlers like Google and Bing to better understand which content is important, which is a subsidiary, which is for navigation, and so on.
